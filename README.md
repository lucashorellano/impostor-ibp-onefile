
# IBP LatAm – El Impostor (PWA) · v3

## Cambios de esta versión
- **Temas** limitados a 10 palabras: Mate, Sushi, Harry Potter, Fast & Furious, Los Simpson, Johnny Depp, George Clooney, Extraterrestre, Paris, New York.
- **Sin temas vagos para impostor**: impostores juegan a ciegas (topic vacío).
- **Múltiples impostores**: 1 impostor cada 4 jugadores (mínimo 1).
- **Puntaje**: acertar = +1 a cada tripulante; fallar = +2 a cada impostor.
- **Compatibilidad GitHub Pages**: service worker y assets con rutas relativas.

## Cómo publicar
1) Subí **todo el contenido** de esta carpeta a un repo (raíz).
2) Activá **GitHub Pages** → Source: Deploy from a branch → Branch: `main` / `/root`.
3) Esperá 1–2 min y abrí la URL `https://usuario.github.io/tu-repo/`.

## Uso rápido
1) Crear sala (modo **Links** por defecto, hasta 30 jugadores).
2) Asignar roles y tema → Copiá los **links privados**.
3) Timer de respuestas → Votación → Calcular acusado → Puntajes.

## Notas
- Si la app no refleja cambios, hacé **Hard Reload** (Ctrl+F5 / Cmd+Shift+R) para limpiar el cache del service worker.
